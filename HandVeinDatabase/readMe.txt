This directory consists of the left hand images data. 

Act directory -> After having squeezed an elastic ball repetitively (closing and opening the fist) for one minute,

Bag directory -> keeps data collected after having carried a bag weighting 3 kg for one minute,

Ice directory -> keeps data collected after having cooled the hand by holding an ice pack on the surface of the back of the hand,

Norm directory -> keeps data collected under normal conditions.
